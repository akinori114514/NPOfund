## 🤔 これは何？
日本の祭りや文化財を「寄付」ではなく**「投資」**によって守る、新しいWebプラットフォームのプロトタイプです。

少子高齢化で資金不足に悩む文化財に対して、AIによる透明な監査システムと、リターン（体験）を伴う投資モデルを導入することで、持続可能な資金循環を生み出すことを目指しています。
