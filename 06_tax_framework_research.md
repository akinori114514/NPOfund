# NPOファンド × 税制優遇フレームワーク — リサーチレポート

> **目的**: NPOファンド構想が、既存の寄附金控除・ふるさと納税等の税制優遇とフィットするか検証する

---

## 1. 現行制度マップ

### 1-1. 寄附金控除（所得税法 §78、地方税法 §37の2）

| 項目 | 内容 |
|------|------|
| **対象法人** | 認定NPO法人、特定公益増進法人（公益財団・社団、学校法人、社会福祉法人等） |
| **個人：所得控除** | （寄附額 − 2,000円）→ 所得から控除。上限 = 総所得の40% |
| **個人：税額控除** | （寄附額 − 2,000円）× 40% → 税額から控除。上限 = 所得税額の25% |
| **住民税** | 条例指定団体なら追加で最大10%控除（都道府県4% + 市区町村6%） |
| **法人** | 特別損金算入限度額あり（一般枠とは別枠） |

> **ポイント**: 個人は **所得控除 or 税額控除を選択可能**。多くの場合、税額控除の方が有利。NPOファンドが認定NPO法人または公益財団法人の形態を取れば、寄附者にこのメリットを提供できる。

### 1-2. ふるさと納税（地方税法 §37の2 特例分）

| 項目 | 内容 |
|------|------|
| **仕組み** | 自治体への寄附。**実質負担2,000円**で残額が所得税+住民税から控除 |
| **3段階控除** | ①所得税還付 ②住民税基本分(10%) ③住民税特例分(上限:住民税所得割の20%) |
| **返礼品** | 寄附額の30%以下（地場産品に限る） |
| **2025年改正** | 仲介サイトのポイント付与禁止（10月〜） |
| **2026年改正** | 地場産品基準の厳格化、高所得者への上限検討 |

> **NPOファンドとの接点**: 一部のコミュニティ財団（しみん基金KOBE、よこはま夢ファンド等）は、ふるさと納税の受け皿として機能している。自治体と連携すれば「ふるさと納税 → NPOファンドの対象事業」というルートが作れる可能性がある。

### 1-3. 企業版ふるさと納税（地方創生応援税制）

| 項目 | 内容 |
|------|------|
| **控除率** | 損金算入(約30%) + 税額控除(60%) = **最大約9割軽減** |
| **対象** | 内閣府認定の地域再生計画に基づく事業への寄附 |
| **下限** | 10万円〜 |
| **制約** | 返礼品禁止, 本社所在地の自治体には不可 |
| **期限** | **2028年3月まで延長**（2025年度改正で3年延長） |
| **NPOの関与** | 自治体の地域再生計画の実施主体としてNPOが参画可能 |

> **企業から見ると実質負担1割**。祭り・文化財保護を地域再生計画に位置付ければ、企業からNPOファンド対象事業への資金誘導が可能になる。

### 1-4. 宗教法人の税制

| 項目 | 内容 |
|------|------|
| **法人税** | 宗教活動収入は**非課税**（法人税法上「公益法人等」） |
| **収益事業** | 34種の指定事業は課税、ただし軽減税率 + みなし寄付金20%損金算入 |
| **個人からの寄付** | ⚠️ **原則として寄附金控除の対象外** |
| **例外** | 財務大臣指定（文化財修復等）or 特定公益増進法人認定時のみ |

> **重要**: 祭りを支える神社・寺院への個人寄付は、現行法では**税制優遇がほぼない**。これはNPOファンドが解決すべき構造的ギャップ。

### 1-5. SIB / PFS（成果連動型民間委託）

| 項目 | 内容 |
|------|------|
| **法的根拠** | 単独法なし。官民連携・委託契約の枠組みで運用 |
| **仕組み** | 投資家が資金拠出 → 成果達成で行政が元本+利息を償還 |
| **税制** | 投資としての性格 → 寄附金控除の直接適用は不明確 |
| **動向** | 山口FG、三井住友FGなど金融機関が主導。2025年以降拡大基調 |

### 1-6. コミュニティ財団

| 項目 | 内容 |
|------|------|
| **形態** | 多くが**公益財団法人** → 寄附金控除の対象 |
| **特徴** | 「マンション型」：寄付者が基金を設立、財団が運用・助成先選定 |
| **日本の規模** | 2023年12月時点で**48財団** |
| **事例** | 大阪コミュニティ財団（1991年〜）、東京コミュニティ財団、あいちコミュニティ財団等 |

### 1-7. 🆕 新公益信託制度（2026年4月施行）

| 項目 | 内容 |
|------|------|
| **受託者** | 信託銀行だけでなく**NPO法人・公益法人・個人**も可能に |
| **信託財産** | 金銭だけでなく**不動産・有価証券・知的財産**も可 |
| **事業範囲** | 助成だけでなく事業運営（フリースクール、美術館等）も可 |
| **税制優遇** | 公益法人と概ね同等：寄附金控除、譲渡所得非課税（承認時）、信託財産からの所得非課税 |
| **ガバナンス** | 内閣府/都道府県による認可制に統一 |

> **⚡ NPOファンドにとって最大のゲームチェンジャー**。2026年4月以降、NPO法人自体が受託者として公益信託を設定でき、多様な資産を受入れ、かつ寄附者に税制優遇を提供できるようになる。

### 1-8. 認定NPO法人（PST要件）

認定NPO法人になるための**パブリックサポートテスト（PST）** 3基準のいずれか：

1. **相対値基準**: 寄附金収入が経常収入の**20%以上**
2. **絶対値基準**: 年平均**100人以上**から各**3,000円以上**の寄附
3. **条例個別指定**: 自治体条例で住民税控除の対象として指定

> NPOファンドの「月1,000円から参加」モデルは、PST絶対値基準（年100人×3,000円以上）をクリアしやすい設計。月1,000円 × 12ヶ月 = 年12,000円 > 3,000円。

## 2. フィット仮説の検証

### 仮説A：宗教法人への寄付の「税制ギャップ」を埋める

```
【現状の問題】
  個人 → 神社（祭り支援）に寄付  →  寄附金控除 ✗ 対象外
  個人 → 認定NPO法人に寄付      →  寄附金控除 ✓ 最大50%還元

【NPOファンドの解決策】
  個人 → NPOファンド（認定NPO/公益財団）に寄付 → 寄附金控除 ✓
       → NPOファンドが市場評価+AI監査で祭りに配分

  つまり、宗教法人への直接寄付では得られない税制優遇を、
  NPOファンドを中間に入れることで寄付者に提供できる。
```

**検証結果: ✅ 強くフィットする**。宗教法人への直接寄付が控除対象外であること自体が、NPOファンドの存在意義を強める。

---

### 仮説B：ふるさと納税の仕組みとの併用

```
【スキーム案】
  自治体が「文化財・祭り保存事業」を地域再生計画に組み込む
       ↓
  個人 → ふるさと納税 → 自治体 → NPOファンド対象事業に配分
  企業 → 企業版ふるさと納税 → 自治体 → 同上（実質負担1割）
```

**検証結果: ✅ フィットする**（ただし自治体との連携が前提条件）。特に企業版ふるさと納税は、企業の実質負担が約1割と極めて有利。2028年まで延長されたことも追い風。

---

### 仮説C：新公益信託制度（2026年4月〜）の活用

```
【スキーム案】
  NPOファンド運営主体（NPO法人）が「公益信託」を設定
       ↓
  寄附者 → 公益信託に寄附 → 寄附金控除 ✓
       → 信託財産をAI監査 + 市場評価で運用・配分
       → 不動産・有価証券等の現物寄附も可能（譲渡所得非課税）
```

**検証結果: ✅ 極めて有望**。NPOファンドのコンセプトと新公益信託制度は思想的にも構造的にもフィットする。

## 3. 推奨実装シナリオ（3案）

### シナリオ①: 公益財団法人型（堅実路線）

- **メリット**: コミュニティ財団の実績多数、寄附金控除の枠組みが確立済み
- **課題**: 公益認定の取得（内閣府の審査）に時間がかかる

### シナリオ②: 新公益信託型（2026年4月〜）

- **メリット**: 制度が新しくスキーム設計の自由度が高い、現物資産も受入可能
- **課題**: 2026年4月施行後の運用実績がまだない

### シナリオ③: 自治体連携ハイブリッド型

- **メリット**: ふるさと納税の圧倒的知名度、企業版の9割控除
- **課題**: 自治体の協力が不可欠、地域再生計画の認定手続き

## 4. 結論と次のステップ

### 結論

> **NPOファンドは、現行税制と極めてフィットする。** 特に以下の3点が決め手：
>
> 1. **宗教法人への直接寄付が控除対象外**という構造的ギャップを、NPOファンド（認定NPO/公益財団）が中間に入ることで解消できる
> 2. **2026年4月の新公益信託制度**は、NPOファンドの「多様な資産を集めてAI評価で配分する」モデルそのもの
> 3. **企業版ふるさと納税**（実質負担1割）との連携で、企業資金の大規模誘導が可能

### 要検討事項

| 項目 | 具体的な論点 |
|------|-------------|
| **法人形態の選択** | 公益財団法人 vs 認定NPO法人 vs 新公益信託 — どれを主軸にするか |
| **PST充足計画** | 認定NPO法人を目指す場合、年100人×3,000円の実績をどう積むか |
| **自治体パートナー** | ふるさと納税連携を狙う場合、どの自治体と組むか |
| **AI監査の法的位置づけ** | 信託財産の運用にAI監査を組み込むことの法的整理 |
| **「投資」と「寄附」の線引き** | ファンドがリターンを返す場合、寄附金控除対象外になるリスク |
